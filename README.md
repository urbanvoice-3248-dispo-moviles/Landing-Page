# UrbanVoice — Landing Page

> Red ciudadana de seguridad colaborativa para Lima Metropolitana.

---

## Descripción del proyecto

**UrbanVoice** es una plataforma de seguridad ciudadana que conecta a los vecinos de Lima con los cuerpos de seguridad (Bomberos, Policía Nacional, Serenazgo, SAMU). Esta landing page presenta sus funcionalidades principales y dirige al usuario a descargar la app móvil.
